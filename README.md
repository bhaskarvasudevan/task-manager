# Task Manager Application

A full-stack task management application with microservices architecture using Java 21.

## 🏗️ Architecture

- **Backend**: Java 21 Spring Boot REST API with Virtual Threads
- **Frontend**: React 18 SPA
- **Database**: MS SQL Server 2022
- **Cache**: Redis 7
- **API Gateway**: Spring Cloud Gateway
- **Web Server**: Nginx
- **CI/CD**: GitHub Actions
- **Deployment**: Docker on Hetzner Cloud

## ✨ Features

- ✅ Full CRUD operations for tasks
- ✅ Redis caching for improved performance
- ✅ Java 21 Virtual Threads for better concurrency
- ✅ ZGC (Z Garbage Collector) for low-latency
- ✅ Responsive React UI
- ✅ API Gateway for routing
- ✅ Automated CI/CD pipeline
- ✅ Docker containerization
- ✅ Health checks and monitoring

## 📋 Prerequisites

- Java 21+
- Node.js 18+
- Maven 3.6+
- Docker & Docker Compose
- GitHub Account
- Hetzner Cloud VM (for deployment)
